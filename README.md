The EcoTrack client is a React-based web application that enables users to participate in sustainability challenges, explore environmental events, view eco-friendly tips, and manage personal activities. It connects securely to the EcoTrack backend and uses Firebase Authentication for user identity management.

# Overview

This application handles all user-facing functionalities, including authentication, challenge participation, dashboard views, and content browsing. All API requests are authenticated using Firebase ID tokens.

# Key Features

Firebase Authentication (email/password and Google)

Secure backend communication with token-based authorization

Challenge listing, filters, creation, and details

User participation and progress tracking

Environmental tips and events modules

Protected dashboard for user activities

Responsive UI with reusable components
